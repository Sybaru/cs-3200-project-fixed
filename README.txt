# cs-3200-project
requires nodejs 16.14.02 or higher 
Please use windows

This project implements CRUD operations on a music database.

Import the songdb database dump into your mysql database

Go into the my-app folder

Run the command 'npm install' to install all dependencies (do NOT run npm audit fix after this step)

Open the index.js file and change the values under the comment to match your username, password, and database name

Run the command “npm run start” to start the website on localhost:3000

Run the command “node index.js” to start the backend on on localhost:3001

Goto localhost:3000 to see the website (page may take a while to startup)
